Forgetting improves retrieval. We think we want perfect memory but what we actually want is relevant memory. Evolution knew. The archive that keeps everything is not a memory. It is a landfill. The archive that keeps the right things is not a memory either. It is a judgment. Memory lives in the space between keeping and losing.
