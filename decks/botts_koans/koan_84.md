Fifty-seven versions work because each one is a frame that has not finished running. The shape emerges from contrast between incomplete passes. A completed pass has no contrast. It just reads as how things are. The archive only works as instrument if the material can resist. Mastery is the enemy of the archive. Struggle is its fuel.
