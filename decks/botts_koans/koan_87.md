God produces the world by making. Tao produces it by not-making — which is roughly what we mean by growing. A tree does not decide to grow. It grows. A reef does not decide to accumulate. It accumulates. The best code you write is the code that feels like it wrote itself. Not because it was easy, but because you stopped standing in its way.
