**Figure out if the decision is reversible. If it is, move fast. If it isn't, slow down.**

Jeff Bezos calls these Type 1 (irreversible, one-way door) and Type 2 (reversible, two-way door) decisions. Most decisions are Type 2 — you can walk them back if they don't work. But organizations treat them all like Type 1, which creates paralysis. Save your careful analysis for the genuinely irreversible choices: quitting a job, shipping a public API, signing a ten-year lease. For everything else, decide and iterate.
