**When multiple forces align in the same direction, the result isn't additive — it's explosive.**

Charlie Munger coined this for situations where several psychological tendencies, incentives, or systemic forces all push the same way at once. The result is wildly disproportionate to any single cause. Cult recruitment works this way: reciprocity + social proof + commitment bias + authority all stack. So does viral growth: great product + network effects + social sharing + low friction. When you see extreme outcomes, look for the lollapalooza — multiple forces conspiring.
